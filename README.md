# COGITO — 지식 탐험

**https://blindso.github.io/cogito-live/**

현실에서 출발해 개념·관계(근거 포함)·반례·현실 신호·가설까지 펼쳐 이해를 돕는 지식 탐험 도구.
5,000개 이상의 큐레이션 주제에 즉시 답하며, 주제는 매일 자동으로 늘어납니다.

- 계정 없이 바로 탐험(브라우저 저장) · 로그인하면 탐험 일지가 계정에 기록됩니다.
- 준비되지 않은 주제는 지어내지 않고 정직하게 "준비 중"으로 안내합니다.

이 저장소는 **빌드된 정적 사이트(공개 배포용)**입니다. 소스는 비공개입니다.
